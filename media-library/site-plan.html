Overview

Purpose

This website enables anyone to easily list their favorite movies. Users will be able to reliably sort by genre, rating, or progress.

Audience

The audience will be anyone who hopes to keep track of their favorite movies in a way that is easy to use.

Dynamic elements

The dynamic elements for the site will include showing media items dynamically as objects inside an array, an event listener tat reacts to user input and filter the media list, and sorting by rating, title, or type.
